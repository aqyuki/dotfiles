## 🌲 Environment

- OS : [Arch Linux](https://archlinux.org/)
- Shell : [fish](https://archlinux.org/)

## 💡 Installation

```bash
git clone https://github.com/aqyuki/dotfiles-linux.git .dotfiles
cd .dotfiles
./install.sh
```
